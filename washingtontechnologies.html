Washington Technologies
Washington Technologies was my first entry for FBLA's computer game and simulation programming event. I developed it during my junior year of high school. It didn't work at the state contest (their computers did not support Open GL), so I got last place. I made the game with jMonkey Engine which is an awesome open source 3D game engine for Java. If you want to play it, right click one of the links below and download.
